### 1. 🥇 超微电脑 SMCI — AI服务器"核心集成商"

#### 第一性原理变量
AI芯片出货量暴增 → AI服务器需求暴增 → Blackwell架构升级 → 单台服务器价值量提升

#### 定量逻辑（具体数据）

| 环节 | 数据 | 来源 |
|:----|:----|:-----|
| **营收增速** | FY2025营收**$236亿** → FY2026目标**$360亿**(+52%) | Yahoo Finance/公司指引 |
| **Q3 FY2026营收** | 未达预期但AI服务器订单积压创历史新高 | CNBC报道 |
| **AI服务器营收占比** | **>70%**（其中Blackwell架构DLC液冷服务器占比快速提升） | 长江证券研报 |
| **毛利率** | ~12-14%（偏低但规模效应改善中） | 财报数据 |
| **PE** | **21.4x** / **Fwd PE 12.5x** | yfinance验证 |
| **核心客户** | NVIDIA、微软Azure、亚马逊AWS、谷歌GCP等超大规模云厂商 | 多来源 |

#### 定价错误的原因（为什么被低估）
SMCI被**Hindenburg Research做空**后，市场将其按"传统服务器代工商"估值(PEG<1)：
- 做空指控：关联交易、出口管制风险
- 市场误解：认为AI服务器是"销量暴增但利润率极低"的生意
- 管理层动荡：审计问题导致年报延迟（已解决）

**实际真相**：SMCI不是纯代工，而是在**DLC液冷服务器**领域拥有核心技术壁垒：
- 每台NVIDIA GB200 NVL72需要SMCI的液冷解决方案
- DLC方案比风冷降低40%功耗，是超大规模数据中心的必选
- Blackwell Ultra/Rubin架构下，液冷渗透率将从30%→90%+

#### 增长催化剂
1. FY2026全年$360亿指引 → 翻倍空间
2. DLC液冷出货占比从20%→70%+（改善毛利率）
3. 审计问题完全解决 → 机构重新入场

### 2. 🥇 美光科技 MU — HBM存储"超级周期"

#### 第一性原理变量
AI芯片算力每代翻倍 → HBM带宽成为瓶颈 → HBM3e→HBM4每代容量翻倍+单价翻倍

#### 定量逻辑（具体数据）

| 环节 | 数据 | 来源 |
|:----|:----|:-----|
| **HBM营收FY2025** | ~$100亿（首次成为最大收入来源） | Micron财报 |
| **HBM营收FY2026** | 预计$200亿+（翻倍） | 多家分析师预测 |
| **HBM4量产** | 2026年H2开始量产，密度比HBM3e翻倍 | Micron官方 |
| **NVIDIA供应商地位** | 美光是NVIDIA HBM3e第三家认证供应商（2024下半年才打入） | 多家报道 |
| **FY2025 total revenue** | **$351亿（+196% YoY）** | yfinance验证 |
| **Fwd PE** | **8.6x**（按周期股估值，但HBM是结构性增长） | yfinance验证 |
| **CapEx** | $200亿投资HBM产能扩张 | Medium/techovedas |

#### 定价错误的原因
市场用"存储周期股"的框架给MU估值（Fwd PE 8.6x）：
- 传统DRAM/NAND是周期波动（3年一轮的涨跌）
- 但**HBM是结构性增长**：HBM3e→HBM4→HBM4e，每次升级容量×2、带宽×2、单价×1.5-2x
- 美光从0→成为NVIDIA第三供应商，证明技术追赶成功

**关键转折点**：美光Q1 FY2026营收创历史新高，HBM4已提前锁定NVIDIA订单。

#### AI算力→HBM需求的链式推导
```
1 块 Blackwell GPU = 8 颗 HBM3e
1 块 Rubin GPU = 12 颗 HBM4（预估）
100万块GPU = 800-1200万颗 HBM
HBM单价：$30-50/颗 → 这是一个$300-600亿的市场（仅NVIDIA一家）
```
美光在其中占据25-30%份额 → 约$75-180亿 HBM营收。

### 3. 🥇 星座能源 CEG — AI核电"最大受益者"

#### 第一性原理变量
单座AI数据中心=100万家庭用电量 → 美国电网无法新增足够电力 → 核电重启是唯一大规模零碳基荷方案

#### 定量逻辑（具体数据）

| 环节 | 数据 | 来源 |
|:----|:----|:-----|
| **2025净利润** | **同比增长+64%** 🔥 | Reuters报道 |
| **与Meta核电协议** | 2025年10月签定**核电购电协议(PPA)**，为Meta AI数据中心供核能 | Guzman分析报告 |
| **收购Calpine** | 2026年收购天然气发电商Calpine（$164亿），扩大清洁能源版图 | Stock Titan |
| **核电装机量** | 美国最大核电运营商，运营**14座核反应堆**，占全美核电15%+ | 公司官网 |
| **PE** | **21.9x** / **Fwd PE 18.5x** | yfinance验证 |
| **数据需求驱动** | 2025年数据中心电力采购量同比增长300%+ | Forbes报道 |

#### 定价错误的原因
市场仍按"传统公用事业"给CEG估值（20x PE）：
- 传统公用事业增长0-3%/年 → PE 15-20x合理
- 但CEG在AI核电协议推动下，**增长中枢已升至15-20%/年**
- 这本质是**AI基础设施基础设施**，不再是传统电力股

#### AI数据中心电力需求的具体推导
```
1个AI数据中心 = 500MW-1GW电力需求
到2030年美国AI数据中心新增电力需求 = 200-300TWh
相当于：
- 新增20-30座核电站的发电量
- 全美发电量增量的50%+ 
CEG运营14座核电站，新增产能有限 → 电价上涨 → 盈利能力暴增
```

### 4. 🥈 维斯特拉 VST — AI电力"零售溢价"

#### 第一性原理变量
同上。但VST的不同在于：它有**电力零售合同**，能直接以市场价卖给AI数据中心，赚取价差

#### 定量逻辑

| 环节 | 数据 |
|:----|:-----|
| **PE** | **24.5x** / **Fwd PE 13.3x** 🔥 |
| **营收增速** | +43% |
| **核心资产** | 天然气+核能+储能，有直接向数据中心售电的零售牌照 |
| **估值折价** | Fwd PE 13x — 市场担忧天然气价格波动，但AI PPA锁定了电价 |

#### 核心逻辑
VST拥有零售售电牌照和灵活发电资产，可以直接和AI数据中心签定PPA锁定高价，而不像传统发电商只能按电网批发价卖电。这个"零售溢价"是市场没有充分定价的。

### 5. 🥈 哈贝尔 HUBB — 变压器"全球短缺"

#### 第一性原理变量
AI数据中心+电气化→全球变压器需求暴增→产能不足→涨价周期

#### 定量逻辑
| 环节 | 数据 |
|:----|:-----|
| **PE** | **28.7x** / **Fwd PE 22.3x** |
| **订单积压** | **创历史新高**，交货周期从6→24个月 |
| **涨价** | 变压器连续6个季度涨价，累计+30-50% |
| **营收增速** | +11%（受限于产能，不是需求不足） |

#### 核心逻辑
变压器的核心矛盾是**产能瓶颈**：
- 全球主要变压器厂（HUBB、Eaton、GE Vernova）产能已满
- 新建变压器工厂需要3-5年
- AI数据中心+电动车充电+电网升级，三重需求叠加
- HUBB在北美配电变压器市场份额第一，直接受益
