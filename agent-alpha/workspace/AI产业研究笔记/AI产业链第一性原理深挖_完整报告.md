# 一、方法：Serenity的卡脖子(Chokepoint)理论

**核心理念**：在AI光子学供应链中，找到「没有它，GPU就造不出来」的单点瓶颈。这些环节往往被华尔街忽视，因为：
- 公司太小（市值$1B-$50B）
- 技术太专（只有行业内的人懂）
- 尚未被贴上"AI标签"

**7层卡脖子地图**（来自semiconstocks.com追踪的Serenity公开post）：

```
第1层: 原材料(Ga/In/As) → AXTI
第2层: pBN坩埚 → 信越化学(日本)
第3层: InP衬底 → AXTI (最核心卡脖子)
第4层: CW激光器(CPO) → SIVE  
第5层: 光收发器 → AAOI·LITE·COHR
第6层: 测试 → AEHR
第7层: 光纤 → GLW
```

**第一性原理分析：**
- AI GPU必须装在服务器里才能工作 → SMCI是NVIDIA GB200/NVL72的核心合作商

- **风险**：Hindenburg阴影、毛利率承压
