### 🥇 标的1：超微电脑 SMCI

#### 第一性原理变量 → 需求保证的链式推导

```
变量：AI训练/推理算力需求每12-18个月翻10倍
  ↓
GPU从H100→B200→Rubin，单GPU功耗从350W→1500W+
  ↓
单台服务器从8GPU→72GPU(NVL72)，功耗从3kW→130kW+
  ↓
传统风冷散热无法支撑 → DLC液冷从"可选"变"必选"
  ↓
SMCI是全球最早大规模量产DLC液冷AI服务器的厂商（技术壁垒）
  ↓
超大规模云厂商(AWS/Azure/GCP)的液冷服务器订单向SMCI倾斜
  ↓
SMCI营收: FY24 $150亿 → FY25 $236亿(+57%) → FY26指引$360亿(+52%)
```

#### 数据支撑

| 环节 | 具体数据 | 数据来源 |
|:----|:--------|:--------|
| **AI服务器营收占比** | **>70%**（FY2025），且仍在提升 | 长江证券研报 |
| **DLC液冷渗透率** | 从FY24的20%→FY26预期的70%+ | CNBC报道 |
| **FY26营收指引** | **$360亿**（管理层的保守指引，分析师预期更高） | Yahoo Finance |
| **Q3 FY26业绩** | 营收略miss但AI积压订单创历史新高 | CNBC、Earnings Call |
| **PE** | **21.4x** — 按传统服务器代工商估值 | yfinance |
| **Fwd PE** | **12.5x** — 利润增速远未被消化 | yfinance |
| **毛利率** | 12-14%（偏低，但DLC方案毛利率比风冷高5-8pct） | 财报 |
| **客户集中度** | 前3大客户(AWS/Azure/GCP)占比>50% | 财报披露 |

#### 为什么存在定价错误（即为什么市场没发现）

Hindenburg Research在2024年8月发布做空报告后：
1. **做空指控**：关联交易、出口管制风险、审计问题
2. **审计延迟**：年报从8月推迟到11月，引发退市恐慌
3. **市场思维定势**：认为AI服务器是"量大利薄"的代工生意

**实际三点证伪：**
- ✅ 审计已完成，无退市风险
- ✅ DLC液冷技术有壁垒(100+专利)，不是简单组装
- ✅ Blackwell Ultra/Rubin架构下，液冷是唯一方案，SMCI是少数能量产厂商

#### AI利润弹性量化

```
FY26 $360亿营收 × 13%毛利率 = $46.8亿毛利
扣除$20亿费用后 ≈ $26.8亿净利 → Fwd PE ≈ 360亿/26.8亿 = 13.4x
如果毛利率改善到15%（DLC占比提升）→ 净利$34亿 → PE仅10.6x
```

### 🥇 标的2：美光科技 MU

#### 第一性原理变量 → 需求保证的链式推导

```
变量：AI大模型参数量每18个月翻100倍（GPT-3 175B→GPT-4 1.8T→GPT-5预估10T+）
  ↓
模型参数需要存储在GPU的HBM中（HBM是AI芯片的"高速缓存"）
  ↓
H100：80GB HBM → B200：192GB HBM → Rubin：288GB HBM+
  ↓
每代升级：容量翻倍 × 带宽翻倍 × 单价涨30-50%
  ↓
DRAM厂从"按晶圆产能定价"转向"按HBM颗粒定价"（价值量暴增）
  ↓
美光2024下半年打入NVIDIA HBM3e供应链（此前只有SK海力士和三星）
  ↓
HBM4量产（2026H2）：密度翻倍 → 美光份额从0→25-30%+
  ↓
美光HBM营收：FY25 ~$100亿 → FY26预估$200亿+
```

#### 数据支撑

| 环节 | 具体数据 | 数据来源 |
|:----|:--------|:--------|
| **HBM营收FY25** | **~$100亿**（首次成为最大收入来源） | Micron财报 |
| **HBM营收FY26** | 分析师预估**$200亿+**（翻倍） | 多家券商 |
| **HBM4进入NVIDIA链** | 2025年底确认，2026H2量产 | Micron官方 |
| **CapEx** | **$200亿**投资HBM产能（集中在爱达荷州新厂） | techovedas |
| **FY25总营收** | **$351亿（+196% YoY）**🔥 | yfinance |
| **Fwd PE** | **8.6x** — 按周期股估值（严重低估） | yfinance |
| **库存** | 库存周转天数降至历史低位（HBM供不应求） | 财报 |

#### 为什么存在定价错误

市场用"存储周期股"框架给MU估值（Fwd PE 8.6x）：
- 传统DRAM/NAND：3年一轮周期，涨价→扩产→跌价→减产→涨价
- **HBM完全不同**：不是容量周期，是**技术升级周期**
- HBM3e→HBM4→HBM4e，每一代都是容量×2、带宽×2、单价×1.5-2x
- 这不是周期，是**AI拉动的结构性增长**

**类比：** 市场给周期股8x PE，但HBM这个业务本身应该按成长股25-30x PE估值。如果拆分估值，HBM业务按$100亿营收×30%净利率×25x PE = $750亿，存储按$250亿营收×10%净利率×8x = $200亿，合计$950亿。当前市值$1000亿附近→隐含HBM几乎没有溢价，这是最大错误。

#### HBM市场量化

```
1颗B200 GPU = 8颗HBM3e（每颗24GB）
100万颗B200 = 800万颗HBM3e
HBM3e单价约$30-50/颗 → 仅NVIDIA一家每年需求 = $300亿+
到2027年HBM总市场 = $600-1000亿
美光占25%份额 = $150-250亿营收
如果净利率30% = $45-75亿净利
按成长股20x PE = $900-1500亿
```

### 🥇 标的3：星座能源 CEG

#### 第一性原理变量 → 需求保证的链式推导

```
变量：单座AI数据中心用电量 = 30万-100万家庭用电量
  ↓
1座500MW AI数据中心：年用电量 = 4.4TWh
到2030年美国AI数据中心新增电力需求 = 200-300TWh
  ↓
美国电网已满负荷，新建输电线路需5-10年
新增天然气发电不满足科技公司碳中和承诺
  ↓
核电成为唯一可行的大规模零碳基荷方案
  ↓
微软2024年底签Three Mile Island核电重启协议（20年PPA）
Meta2025年10月签CEG核电协议（为AI数据中心供电）
谷歌2025年底签Kairos Power的小型核反应堆协议
  ↓
CEG是美国最大核电运营商（14座反应堆，占全美核电15%+）
核电资产年限从60年延长到80年（政策允许）
收购Calpine（天然气）补齐清洁能源版图
  ↓
CEG净利：2024年→2025年(+64%)→2026年预计再+30%+
```

#### 数据支撑

| 环节 | 具体数据 | 数据来源 |
|:----|:--------|:--------|
| **净利增速2025** | **+64%**🔥 | Reuters |
| **与Meta核电协议** | 2025年10月签定，为Meta AI数据中心专供核能 | Guzman分析 |
| **收购Calpine** | **$164亿**收购天然气发电商（2026完成） | Stock Titan |
| **核电装机** | **14座反应堆**，总装机~18GW | 官网 |
| **数据中心电力采购** | 2025年同比增长300%+ | Forbes |
| **PE** | **21.9x** / **Fwd PE 18.5x** | yfinance |
| **可再生能源占比** | 90%+（碳中和目标下的AI首选能源供应商） | 公司报告 |

#### 为什么存在定价错误

市场给公用事业股PE 20x是"合理"的，但：
- 传统公用事业增长=0-3%/年（人口增长+GDP增长，自然增长天花板）
- CEG在AI协议推动下，增长已升至15-20%/年
- 假设增长20%/年 → PE 20x → PEG=1（合理估值）
- 但市场还没把AI核电协议的价值计入估值模型

**关键转折：** 微软2024签Three Mile Island → Meta2025签CEG → 其他科技巨头必跟。AI核电是"刚起步的超级趋势"，不是"过气题材"。

### 🥈 标的4：维斯特拉 VST

#### 第一性原理变量 → 需求保证的链式推导

```
变量：AI数据中心需要24×7不间断供电
  ↓
电网批发价波动大（受天然气价格影响）
  ↓
AI数据中心需要稳定、可预测的电价（3-5年PPA锁定）
  ↓
拥有零售售电牌照的电力商可以直接和AI数据中心签PPA
  ↓
VST拥有德克萨斯州最大零售售电牌照
可以直接把天然气/核电/储能打包卖给AI数据中心
  ↓
VST签的AI PPA价格 = 电网批发价 + 零售溢价（约20-30%溢价）
  ↓
VST营收增速+43%，PE 24.5x / Fwd PE 13.3x
```

#### 为什么有额外溢价
VST不同于一般发电商（如NRG、AES）：它有**零售售电牌照和零售客户群**，可以直接和AI数据中心签购电合同，赚取零售溢价。这个"中间商"模式让它在AI电力需求中有独特的议价能力。
