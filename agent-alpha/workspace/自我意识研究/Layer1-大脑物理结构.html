一、灰质 vs 白质：一句话区分

想象你切开一个核桃：外面的皱皮是灰质（算盘），里面浅色的坨是白质（电线）。

记住这一句就够了：灰质做计算，白质做传输。结构网络看白质纤维束，功能网络看灰质之间信号有多同步。

二、先定坐标：前后·上下·内外

先搞清楚一个问题——为什么「背侧(dorsal)」等于「头顶方向」？因为人类的祖先趴着走，背朝上肚朝下。我们直立以后，原来的「背」变成了「头顶」，「腹」变成了「下巴」。神经解剖学保留了趴着的叫法。所以你看到的：

背侧(Dorsal) = 头顶方向  |  腹侧(Ventral) = 下巴方向

前/嘴侧(Anterior/Rostral) = 额头方向  |  后/尾侧(Posterior/Caudal) = 后脑勺方向

外侧(Lateral) = 耳朵方向  |  内侧(Medial) = 头顶中线（两半球之间的缝）

为什么背=头顶、腹=下巴？因为人类的老祖宗是趴着走的——背朝上，肚子朝下。我们直立以后转了 90°，但神经解剖学保留了原来的叫法。
读论文时最简单的翻译：Dorsal = Superior = 上，Ventral = Inferior = 下，Anterior = 前，Posterior = 后。两组词完全等价，只是习惯不同——MRI论文用 Superior/Inferior 多，动物实验用 Dorsal/Ventral 多。

三、六个地标：用你的头做参考系

1. 前额叶皮层（PFC）—— 大脑CEO的三个办公室

2. 脑岛（Insula）—— 藏在褶皱里的身体告警台

3. 扣带回（Cingulate）—— 骑在胼胝体上的马鞍

4. 角回（Angular Gyrus）—— 概念的接线员

5 & 6. 海马 & 杏仁核 —— 埋在深处的记忆索引和烟雾报警器

最后一个技巧：读任何脑成像论文时，把「BA N」翻译成位置而不是数字——「BA 24 激活了」→「dACC 位置亮了」→「冲突监控/SN 在工作」。用位置代替数字，论文立刻可读。

快速自测：闭眼用手摸

1. 手放额头 → 后面是PFC。外侧靠上 = DLPFC，正中靠下 = VMPFC。

2. 手指往耳朵眼里推 → 深处是脑岛。前段(AI)是 SN 的第一把交椅。

3. 手沿头顶中线往后摸 → 下面深处是扣带回，骑在胼胝体上。前段 dACC，后段 PCC。

4. 手指放耳朵尖上方 3cm 偏后 → 角回。DMN 的概念接线员。

5. 耳朵眼里再往里推 3cm → 海马（记忆编目）和杏仁核（恐惧报警器）。
