# 关键变量的核心判断

```
变量1: GPU功耗 300W→1000W+   → 液冷必选 → LII 受益
变量2: 集群电耗 100MW→1GW    → 变压器短缺 → HUBB 受益  
变量3: 网络带宽 400G→1.6T    → 光模块爆发 → AAOI/CRDO 受益
变量4: 训练数据 每季翻倍     → HBM/存储 → MU 受益
变量5: 封装密度 2D→3D       → 封装产能瓶颈 → AMKR 受益
```

**最大的认知差：MU的Forward PE仅8.6x** —— 市场完全没有把HBM的超级周期计入价格。$1.06T市值看似巨大，但$24B净利润且仍在翻倍增长，实际PE仅8.6x！
